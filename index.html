Projects

Query Parser

Spring 2020 | Github
Bootstrap, Javascript, Java, MongoDB

The goal of the application is to make a query parser that uses a query language to run against other databases. This application assists users in managing data in databases with RDBMS SQL familiarity. Additionally, this gives users the ability to easily access data from other database systems and the convenience of knowing only one domain specific language.

Carb⟲

Spring 2020 | Github
Augmented Reality, Unity, C#

Carb⟲ is an augmented reality (AR) app that shows the journey of carbon molecules during the Carbon Cycle. This application is intended to solve the abstractness problem in Biology because it can be difficult for students to understand how certain molecules, like carbon, play roles in our everyday world.

Cuisino

Fall 2020
Flutter, Dart, Firebase

Cuisino provides individuals and groups with a "Tinder-style" voting game that helps choose a restaurant based on location, preferences and dietary restrictions. Users can join or create, permanent or temporary, groups to give users a quick and easy way to find restaurants. Cuisino was made with Flutter to be able to create an application that can be developed in both, Android and iOS.
